# Završni rad - JavaFX aplikacija za upravljanje autoservisom

> U razvoju

Ovaj projekt razvijam kao svoj završni rad na trećoj godini Računarstva.

Koristit ću znanje naučeno na kolegiju **Programiranje u jeziku Java** što uključuje:

* JavaFX
* OOP Principi
* JDBC & Datoteke
* Generički tipovi
* Multi-threading
